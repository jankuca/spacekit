Space Kit Framework Documentation

The spacekit framework is a library for composing and translating elements using CSS transforms. The main advantage of the library is that it is compilable in the ADVANCED_OPTIMIZATIONS compilation mode and the VERBOSE analysis level via the Google Closure Compiler and that it is fucking awesome and fast.
